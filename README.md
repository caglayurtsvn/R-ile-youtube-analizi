# R ile Youtube Haber Analizi
Bu projede R dili ve Shiny ile geliştirilen interaktif bir uygulama yer alıyor. Amaç; asgari ücret, Ahmet Minguzzi, deprem, iklim değişikliği gibi konular üzerine sosyal medya yorumlarını analiz etmektir.

## 🔍 Kullanılan Yöntemler
- Kelime Bulutu (Wordcloud)
- Emoji ve Duygu Analizi
- Bigram/Trigram Ağı
- Shiny


## 🧠 Kullanılan Paketler
`shiny`, `tidyverse`, `textclean`, `emo`, `ggplot2`, `leaflet`, `textdata`, `wordcloud2`, `igraph`
